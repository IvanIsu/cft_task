Java-19.0.1 сторонние библиотеки не использовал
для запуска порядок аргументов не важен, главное чтобы сохранялось пара ключ-значение для аргументов(-o values -p values  -files values)
-o путь к результатом(можно указать абсолютный путь либо относительный)
-p префикс
-s простая статистика
-f расширенная статистика
-a дописывать в фаил
-files агрумент для указания читаемых файлов  -files file1.txt,file2.txt
пример запуска
java -classpath ./bin Main -files in1.txt,in2.txt -o \results -p result_ -a -f
